## Cuckoo Hashing

A simple C++ implementation of Cuckoo Hashing which supports single-slot and multi-slots.



### Build

```shell
./make.sh
```



### Run

#### single-slot

```shell
cd build
./CuckooHashing [testCount] [unitsLen]
```

#### multi-slots

```shell
cd build
./CuckooHashingSlots [testCount] [unitsLen] [slotsNum]
```
